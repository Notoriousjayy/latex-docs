\section*{Companion record: official ID summary}

The certificate above is paired with the following official record. Both
share the same module so the typography and palette match.

  \IDField{Holder}{Jordan Suber}
  \IDField{Credential}{Style Hierarchy Custodian}
  \IDField{Credential ID}{LDS-2026-001}
  \IDField{Issued}{2026-05-08}
  \IDField{Authority}{Tooling \& Documentation WG}

\subsection*{Notes on this module}
The \texttt{certificate} environment is deliberately a centred,
self-contained page; the running header is suppressed so the
credentialing artifact reads cleanly. The \texttt{signedby} macro is
a single-signatory variant of the parent's \texttt{LdsSignatureLine}
primitive, composed in rather than reimplemented, exactly as the
architecture rationale prescribes.
