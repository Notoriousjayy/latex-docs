\section*{How to write effective user stories (quick guide)}
\textbf{Template} \emph{As a \underline{<persona>}, I want to \underline{<capability>} so that \underline{<outcome/value>}.}

\medskip
\textbf{BDD Acceptance Criteria} use \textbf{Given/When/Then}:\\
\textbf{Given} the preconditions \quad \textbf{When} the action/event occurs \quad \textbf{Then} the observable outcome happens.

\medskip
\textbf{Quality tags} add small ``pills'' for non-functional goals (Security, Safety, Reliability, Accessibility, Privacy, i18n, etc.).

\medskip
\textbf{Definition of Ready / Done} appear at the bottom of each card to keep scope clear and testable.
