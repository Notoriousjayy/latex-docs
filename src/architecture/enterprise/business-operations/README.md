# Small Business Core Systems Artifact Bundle

This bundle contains separate downloadable artifacts for a small business core systems operating model.

## Files

1. `small_business_core_systems_operating_model.pdf` - master operating model handout.
2. `marketing_lead_generation_system_guide.pdf` - marketing and lead generation system guide.
3. `sales_lead_conversion_system_guide.pdf` - sales and conversion system guide.
4. `operations_fulfillment_system_guide.pdf` - operations and fulfillment system guide.
5. `financial_management_system_guide.pdf` - financial management system guide.
6. `admin_it_security_system_guide.pdf` - administration and IT security system guide.
7. `hr_people_management_system_guide.pdf` - HR and people management system guide.
8. `leadership_strategy_system_guide.pdf` - leadership and strategy system guide.
9. `sop_and_tooling_layer_guide.pdf` - SOP and tooling layer guide.
10. `kpi_and_continuous_improvement_layer.pdf` - KPI and continuous improvement layer guide.
11. `small_business_systems_implementation_checklist.pdf` - implementation checklist.
12. `small_business_core_systems_architecture.puml` - PlantUML architecture diagram source.

The `.tex` source files are included for all PDF handouts.
