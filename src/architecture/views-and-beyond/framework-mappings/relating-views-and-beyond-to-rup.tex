\textbf{To Achieve This RUP View} & \textbf{Use This Views and Beyond Approach} \\
\midrule

Use case view &
Adopt use cases to specify behavior, either associated with any of the views or as part of the documentation beyond views. \\
\addlinespace[0.6em]

Logical view &
Use a module style that shows generalization, uses, and decomposition for structural aspects, and a C\&C style for the runtime aspects. \\
\addlinespace[0.6em]

Implementation view &
Use a module style that contains implementation elements. Use an implementation view to show allocation to development files. \\
\addlinespace[0.6em]

Process view &
Use a style such as the communicating-processes style. \\
\addlinespace[0.6em]

Deployment view &
Use the deployment style, one of the allocation styles. \\
