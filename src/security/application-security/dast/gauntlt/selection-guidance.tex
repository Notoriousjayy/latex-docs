\section*{Selection Guidance}
\begin{itemize}
  \item Use \textbf{Gauntlt} when the team needs readable, version-controlled attack scenarios that wrap approved command-line tools.
  \item Use \textbf{OWASP ZAP} when the control objective is broad web application DAST coverage.
  \item Use \textbf{Nmap}, \textbf{SSLyze}, and \textbf{curl} for deterministic baseline gates.
  \item Use \textbf{SQLMap} only when SQL injection testing is explicitly authorized and carefully scoped.
  \item Consider \textbf{Nuclei} or custom policy-as-code for modern template-based checks where Gauntlt adapter maintenance is a concern.
\end{itemize}
